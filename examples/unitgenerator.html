FieldID C Quantity:
9 CharactersA text modifier in an equipment symbol that identifies the number of items present.
FieldID F Reinforced or Reduced:
3 CharactersA text modifier in a unit symbol that displays (+) for reinforced, (-) for reduced, () reinforced and reduced..
FieldID G Staff Comments:
20 CharactersA text modifier for units, equipment and installations; content is implementation specific.
FieldID H Additional Information:
20 CharactersA text modifier for units, equipment, and installations; content is implementation specific.
FieldID J Evaluation Rating:
2 CharactersA text modifier for units, equipment, and installations that consists of a one-letter reliability rating and a one-number credibility rating.
FieldID K Combat Effectiveness:
5 CharactersA text modifier for units and installations that indicates unit effectiveness or installation capability.
FieldID L Signature Equipment:
1 CharactersA text modifier for hostile equipment; "!" indicates detectable electronic signatures.
FieldID M Higher Formation:
21 CharactersA text modifier for units that indicates number or title of higher echelon command (corps are designated by Roman numerals).
FieldID N Hostile:
3 CharactersA text modifier for equipment; letters "ENY" denote hostile symbols.
FieldID P IFF/SIF:
5 CharactersA text modifier displaying IFF/SIF Identification modes and codes.
FieldID Q Direction: Degrees 4 CharactersA graphic modifier for units and equipment that identifies the direction of movement or intended movement of an object.
FieldID T Unique Designation:
21 CharactersA text modifier for units, equipment, and installations that uniquely identifies a particular symbol or track number. Identifies acquisitions number when used with SIGINT symbology.
FieldID V Type:
24 CharactersA text modifier for equipment that indicates types of equipment.
FieldID W DTG:
16 CharactersA text modifier for units, equipment, and installations that displays DTG format: DDHHMMSSZMONYYYY or "O/O" for on order (see 5.5.2.6).
FieldID X Altitude/Depth:
14 CharactersA text modifier for units, equipment, and installations, that displays either altitude flight level, depth for submerged objects; or height of equipment or structures on the ground.
FieldID Y Location:
19 CharactersA text modifier for units, equipment, and installations that displays a symbol's location in degrees, minutes, and seconds (or in UTM or other applicable display format).
FieldID Z Speed:
8 CharactersA text modifier for units and equipment that displays velocity as set forth inMIL-STD-6040.
FieldID AA Special Headquarters:
9 CharactersA text modifier for units; indicator is contained inside the frame; contains the name of the special C2 Headquarters.

This is an implementation of APP6 (B) / 2525C / 2525D in pure javascript. All images are scalable vector graphics generated from.
In some cases where the standards differ, I have chosen the one that I like the best, main focus has been that all symbol should be possible to produce and that it looks good. The library is also made to follow the way that symbols are created in APP6 (C) / 2525D because this way is more elegant than the old way, but there are differences, however good cartography has always been the main focus. All symbols have been made to be readable down to a size of L=15px.
At the moment APP6(B) ANNEX D / 2525C APPENDIX A (Unit Symbols) and 2525C APPENDIX D (Signals intelligence symbols) and APPENDIX E (Stability operations symbols) / parts of unit symbols for 2525D are implemented in this library.
This page contains no external dependencies, so if you want to use it in a system that isn't connected to the internet, you can just save this single html page.
